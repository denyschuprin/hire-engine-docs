# Welcome to Hire Engine

Hire Engine helps you build a polished **employer profile** for a company and
publish it - along with the company's **open jobs** - to Kununu, plus an
optional **career page** you can share or embed on the company's own website.

This guide walks you through the platform **step by step**. It's written for
everyday use - no technical knowledge needed.

!!! tip "How to use this guide"
    The pages on the left follow the natural order of work, from start to
    finish. If you're testing the platform for the first time, just read them
    top to bottom. If you're looking for one specific thing, use the **search
    bar** at the top.

## The big picture

Working with a company in Hire Engine follows the same simple path every time:

1. **Add the company** - give it a name and a few web addresses.
2. **Collect information** - Hire Engine automatically reads the company's
   website, careers page, Wikipedia and more.
3. **Create the profile** - the system drafts a complete employer profile;
   you review and polish the text.
4. **Approve it** - mark the profile as ready.
5. **Connect Kununu** - enter the login the system uses to publish.
6. **Publish** - push the profile and the jobs to Kununu.
7. **Share the career page** - an optional public page with the profile and jobs.

```
Add company → Collect info → Create profile → Approve → Connect Kununu → Publish → Share
```

## What each part does

| Area | What it's for |
|---|---|
| **Companies** | Every company you work on lives here, with its own profile, jobs and settings. |
| **Sources** | The places Hire Engine reads to learn about the company (website, careers page, Wikipedia, XING). |
| **Profile** | The employer-branding text - drafted automatically, edited by you. |
| **Credentials** | The Kununu logins the system uses to publish on your behalf. |
| **Career Page** | A ready-made public page showing the profile and open jobs. |
| **Jobs** | The company's job postings, collected automatically and published to Kununu. |
| **Users** | (Admins only) Manage who can sign in to Hire Engine. |

!!! note
    Some steps run **in the background** - for example, reading a company's
    website or drafting the profile. You don't have to wait on the screen;
    the status updates on its own when each step finishes.
