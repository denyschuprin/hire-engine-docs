# Managing Users (Admins)

Administrators can create accounts for colleagues, assign their role, and remove
people who no longer need access. This page is only visible to admins - look for
**Users** in the left sidebar.

!!! info "Admins only"
    If you don't see **Users** in the sidebar, your account isn't an
    administrator. Ask an admin to make any changes you need.

## See who has access

The **Users** page lists everyone on your account, showing each person's
**email**, **role**, and whether they're **Active**. Your own row is marked with
a green **You** badge so it's easy to spot.

## Create a user

1. Go to **Users** in the sidebar.
2. In the **Create user** box, enter the person's **email**.
3. Set a **password** (at least 8 characters).
4. Choose a **role** from the dropdown:
    - **Operator**
    - **Agency Admin**
    - **Agency Operator**
5. Click **Create user**.

A green **User created** confirmation appears, and the new person can sign in
right away with the email and password you set.

!!! note "Admins can't be created here"
    For safety, this page can only create the roles above - it cannot create new
    **administrator** accounts.

## Remove a user

1. Find the person in the list.
2. Click the red **trash icon** on their row.
3. Confirm when asked.

The account is removed and that person can no longer sign in.

!!! warning "Admins can't be deleted"
    The delete button is disabled for administrator accounts - they can't be
    removed from this page. This protects you from accidentally locking everyone
    out.
